# SimpleSwitch
不需要图片资源的Switch，参考https://github.com/Leaking/SlideSwitch 和android-support-v7的SwitchCompat，还需完善，多多指教。
###   对SlideSwitch改进有3：
###   1.继承CompoundButton，方便使用
###   2.在OnDraw过程中不创建对象
###   3.优化onTouchEvent和动画的逻辑
